    Install packages

  • Programming languages, e.g. python=2.7, scala
  • Databases and document stores, e.g. mongodb
  • Libraries, e.g. pandas, bokeh, spark

    • $ conda install python=2.7 mongodb pandas bokeh=0.8

    Manage environments


    Write your environment.yml file

    vim environment.yml
     
name: pydata
dependencies:
  - python=2.7
  - mongodb
  - pandas
  - bokeh

    Manage environments


    Create your environment

    $ conda env create

    Manage environments


    Activate your environment

    $ source activate pydata

    Manage environments


    List all the packages in that environment

    $ conda list
    or 
    $ conda env export
